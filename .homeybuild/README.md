# Fresh Intellivent SKY for Homey

Control your Fresh Intellivent SKY fans via Bluetooth using Homey Pro.

## Features
- Read Temperature and Humidity
- Read Fan Speed (RPM)
- Control Fan Speed
- Toggle Boost, Pause, Constant Speed, Humidity, Light/VOC, and Airing modes.
